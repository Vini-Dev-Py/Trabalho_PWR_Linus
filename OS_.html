Sistema Operacional - Unix

Para que seja possível entender o que é o Linux, deve-se entender a base a qual seu sistema operacional funciona: o Unix. Unix é um sistema operacional criado por Kenneth Thompson após um projeto de sistema operacional não ter dado certo. O Unix foi o primeiro sistema a introduzir conceitos muito importantes para SOs como suporte a multiusuários, multitarefas e portabilidade. Além disso, o Unix suporta tanto alterações por linhas de comando, que dão mais flexibilidade e precisão ao usuário, quanto definições via interface gráfica, uma opção normalmente mais prática e menos trabalhosa do que a anterior. Atualmente, uma série de SOs são baseados no Unix, entre eles, nomes consagrados como Gnu/Linux, Mac OS X, Solaris e BSD. Talvez você nem saiba, mas o sistema operacional que roda no caixa eletrônico onde é possível você sacar seu dinheiro, por exemplo, provavelmente é um do tipo Unix.

Diversos fatores podem ser citados ao exemplificar o motivo da popularidade deste sistema em relação a seus “concorrentes”, o Unix propôs um sistema multitarefa, capaz de executar dezenas de processos simultaneamente. Ademais, outra característica do Unix é o suporte a multiusuário. O sistema permite que várias aplicações sejam executadas de modos independente e concorrente por usuários diversos, o que faz com que o compartilhamento seja feito não somente de hardwares, mas também softwares e componentes como discos rígidos e impressoras. Outro fator que com certeza influenciou na popularidade do Unix foi ele ter funcionado sob uma licença livre em seus primeiros anos de vida, tendo sido distribuído gratuitamente para universidades e órgãos governamentais dos Estados Unidos. Apenas depois de algum tempo a licença se tornou proprietária. Contudo, a maioria dos sistemas criados com base no Unix funciona sob um sistema total ou parcial de código aberto. Desse modo, a proliferação do sistema foi impulsionada pela licença livre, principalmente das famílias BSD, Open Solaris e Linux.

O Linux, da mesma forma que outros sistemas operacionais que foram citados acima, é um sistema baseado em Unix criado para desktops, mas que também é usado em servidores, smartphones, tablets e outros tipos de dispositivos, incluindo caixas bancárias. Ao contrário de seus concorrentes mais famosos, o Linux não foi desenvolvido para fins comerciais e seu software e desenvolvimento são feitos em código aberto, o que significa que qualquer pessoa pode criar e distribuir aplicativos para ele. A parte básica do Linux é composta de um kernel, conjunto de instruções que controla como será usado o processador, a memória, o disco e os dispositivos periféricos. É o software presente em todo sistema operacional que dita como o computador deve funcionar. O Kernel Linux foi criado por Linus Torvalds, com a primeira versão oficial lançada em 1991. Para funcionar, porém, também é necessário aplicativos e bibliotecas específicas para eles. Isto significa que um usuário de Linux pode escolher entre diversos aplicativos para executar a mesma função, sejam eles editores de texto, interfaces gráficas ou mesmo prompts de comando.O processo é semelhante à escolha entre Chrome e Firefox: os dois são navegadores, capazes de fazer a mesma coisa, mas de formas e aparência distintas.

O mundo Linux é dominado pela diversidade. Como o código-fonte do programa pode ser modificado livremente, geralmente existem vários aplicativos para cada solução. Existem vários sistemas gráficos no Windows, por exemplo, no Linux. Cada um tem usabilidade, efeitos visuais e recursos exclusivos. Dentre as interfaces disponíveis, UNITY, GNOME Shell e KDE são a maioria das interfaces, e suas alterações trarão experiências diferentes. Existem também Canela, Mate, LXCE, XFCE, Pantheon Shell, OpenBox, Enlightenment, Fluxbox, etc. O mesmo se aplica a outros aplicativos. Para cada segmento de mercado, existem muitas opções. E o programa também pode ser distribuído em um formato incompatível com outros programas. A diversidade do Linux é sua vantagem e seu maior desafio. Com tantas opções, é necessário juntar todas as peças para fornecer uma solução acessível ao usuário. É aí que surge a distribuição Linux.

A distribuição é um projeto desenvolvido para empacotar um conjunto de aplicativos Linux, padrões estabelecidos e assistentes de instalação. Semelhante a um carro, será um fabricante de automóveis que conecta várias peças para criar um carro pronto para uso. Cada atribuição é criada para um perfil de uso. Algumas das finalidades são simplificar o uso de computadores desktop, outras finalidades são usar servidores, com foco em centros de mídia, computadores mais antigos ou ferramentas com interfaces de toque. A escolha dependerá das necessidades. Portanto, para começar a usar o Linux, sua primeira decisão deve ser qual distribuição usar e, em seguida, aproveitar as novas possibilidades. Contar com uma comunidade muito receptiva aos questionamentos dos novos usuários e garantir que, impulsionados pela disseminação e desenvolvimento do conhecimento, devemos cooperar e acompanhar as tendências atuais.

Desde então, o desenvolvimento do sistema operacional Linux tem contado com a colaboração de milhares de desenvolvedores, e novos aplicativos foram adicionados que permitem que o sistema seja usado em vários dispositivos e usos. O Linux está quase em todo lugar. Por exemplo, seu roteador doméstico é igual ao codificador de TV a cabo e provavelmente usa Linux. Os servidores do sistema Linux e outras soluções com licença GPL oferecem suporte principalmente à Internet. Muitos softwares importantes, como servidor Apache Web, ligação DNS, linguagem de programação PHP, Java, Javascript, banco de dados MySQL, PostgreSQL, etc. são outras soluções sob a licença GPL. É melhor rodar no sistema Linux e usar o que sabemos hoje Maneiras de fazer a Web funcionar. Facebook e Netflix são executados em Linux. Assim como a nuvem de computação mais popular do mundo, Amazon AWS. O sistema operacional Android é baseado em Linux, então muitas pessoas pensam que é Linux. No mundo digital dominado pelo software livre, o uso do Linux não é mais discutido. Existe em nossas vidas de uma forma ou de outra. Você pode até se especializar em ferramentas proprietárias, mas com certeza encontrará o Linux em sua carreira. Entender o Linux não é apenas uma vantagem, mas uma jornada impressionante de descoberta. Tudo pode começar na sua área de trabalho.
